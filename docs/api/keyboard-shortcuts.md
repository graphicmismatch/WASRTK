# Keyboard Shortcuts

## Overview

WASRTK provides comprehensive keyboard shortcuts for efficient pixel art creation and animation workflow. This document lists all available shortcuts organized by functionality.

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+N` | New Project | Create a new project with default settings |
| `Ctrl+O` | Open Project | Open an existing WASRTK project file |
| `Ctrl+S` | Save Project | Save current project as .wasrtk file |
| `Ctrl+Shift+S` | Save Animation | Save animation as PNG sequence or GIF file |
| `Ctrl+Shift+R` | Open Reference Image | Import PNG, JPG, GIF, or BMP as reference |
| `Ctrl+Q` | Exit | Quit application |

## Drawing Tools

### Tool Selection
| Shortcut | Action | Description |
|----------|--------|-------------|
| Click Pen Tool | Pen Tool | Select pen tool for pixel-perfect drawing |
| Click Line Tool | Line Tool | Select line tool for straight lines |
| Click Rectangle Tool | Rectangle Tool | Select rectangle tool for shapes |
| Click Circle Tool | Circle Tool | Select circle tool for circular shapes |
| Click Fill Tool | Fill Tool | Select fill tool for area filling |
| Click Eraser Tool | Eraser Tool | Select eraser tool for removing pixels |

### Tool Settings
| Shortcut | Action | Description |
|----------|--------|-------------|
| `[` | Decrease Brush Size | Reduce brush/pen size by 1 pixel |
| `]` | Increase Brush Size | Increase brush/pen size by 1 pixel |
| `Shift+[` | Decrease Brush Size (5) | Reduce brush/pen size by 5 pixels |
| `Shift+]` | Increase Brush Size (5) | Increase brush/pen size by 5 pixels |
| `A` | Toggle Antialiasing | Enable/disable smooth drawing |

## Frame Management

| Shortcut | Action | Description |
|----------|--------|-------------|
| `F` | Add Frame | Create a new empty frame |
| `D` | Duplicate Frame | Create a copy of current frame |
| `Delete` | Delete Frame | Remove current frame |
| `Space` | Play Animation | Start animation playback |
| `Escape` | Stop Animation | Stop animation playback |

## Layer Management

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Up` | Move Layer Up | Move layer up in stack |
| `Ctrl+Down` | Move Layer Down | Move layer down in stack |
| `Ctrl+E` | Flatten Layer | Flatten current layer |

## Reference Controls

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+R` | Reset Reference | Reset reference image position and scale |
| `Ctrl+A` | Toggle Antialiasing | Enable/disable smooth drawing |

## Edit Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` | Undo | Undo last action |
| `Ctrl+Y` | Redo | Redo last undone action |

## Canvas Navigation

| Control | Action | Description |
|---------|--------|-------------|
| `Middle Mouse` | Pan Canvas | Drag to move canvas view |
| `Ctrl+Mouse Wheel` | Zoom Canvas | Scroll to zoom in/out |

## Usage Tips

1. **Learn Core Shortcuts**: Master file operations and tool selection
2. **Use Tool Selection**: Click tool buttons to switch between tools
3. **Practice Frame Navigation**: Use F, D, and Delete for frame management
4. **Master Undo/Redo**: Use Ctrl+Z and Ctrl+Y for mistake correction

This comprehensive keyboard shortcut system enables efficient pixel art creation and animation workflow in WASRTK.
